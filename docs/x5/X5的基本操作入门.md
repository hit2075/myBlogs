## 使用范围

## 不适用场景

- 微信小程序

- 非关系性数据库

- 文档的在线编辑只能支持低版本。chrome42以下的浏览器才嗯那个用，现在已经是80+以上了

- 开发的时候需要连接实例，无法连接服务

## 目录结构

- runtime是核心，是闭源的，进行转义成html后运行， 类似于编译器。是通过UIserver进行编译的，这个目录是不动的。
- Studio- 建模工具
- data-Doc平台的文档中心，附件位置
- Tools Init-数据库初始化工具
- Tools Complie-编译工具
- Log- 日志工具
